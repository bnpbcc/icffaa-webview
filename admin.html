Administracion de contenidos Mini CMS institucional

Edite secciones, noticias, agenda, galeria, popups e imagenes sin tocar el codigo. Para publicar en GitHub Pages, conecte un token de GitHub con permiso de escritura.

Vista publica

No se guarda una clave fija dentro de la web. Para GitHub Pages se usa token del administrador; en servidor dedicado se reemplaza por login real.

Datos generales

Acceso oculto

La web compara hashes, no guarda la contrasena escrita. Esto solo oculta el panel en una web estatica; el guardado real sigue protegido por token de GitHub.
